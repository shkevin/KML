kml.statistics.summary package

KML Summary Statistics Module.

The Streaming Summary Statistics module attempts to define summary statistics on data streams.

kml.statistics.summary.StreamingEntropy module

Streaming entropy wrapper module for C++.

This module is used to wrap the C++ Streaming entropy implemtation.

Example

>>> from kml.statistics.summary.StreamingEntropy import PyStreamingEntropy
>>> sm = PyStreamingEntropy(10, 10)
>>> sm.update(list(range(1, 11)))
>>> print(sm.evaluate())
class kml.statistics.summary.StreamingEntropy.PyStreamingEntropy(window_size: Optional[int] = 100, num_bins: Optional[int] = 0) → None

Bases: object

Class containing the Streaming Entropy implementation in Cython.

Streaming entropy wrapper for the C++ StreamingEntropy class. This code contains the public interface usage for the streaming entropy statistic.

Note

if num_bins is set at 0, the StreamingEntropy class will calculate the number of bins to use based off of the Sturges rule.

Parameters

  • window_size (int, optional) – Desired window size. Defaults to 100.

  • num_bins (int, optional) – Desired number of bins to approximate counts. Defaults to 0.

  • c_SE (StreamingEntropy*) – Pointer to the C++ StreamingEntropy implementation.

  • window_size – Desired window size. Defaults to 100.

  • num_bins – Desired number of bins to approximate counts. Defaults to 0.
